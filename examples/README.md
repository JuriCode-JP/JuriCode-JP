# examples/ — スキーマを満たすサンプル法令

このフォルダには、JuriCode-JPのフォーマット仕様(`docs/format-spec.md`)を**満たすサンプル法令ファイル**を置く。実データではなく、フォーマット参照用の見本である。

## サンプル一覧

| ファイル | 法令・条文 | 構造的なポイント |
|---|---|---|
| [keihou/keihou-article-36.md](./keihou/keihou-article-36.md) | 刑法 第36条(正当防衛) | 2項構造、有名判例リンク、政府公定訳 |

## サンプルファイルの位置づけ

- **フォーマット参照**: 新しい法令データを追加するときの**正規のテンプレート**
- **検証ベンチ**: `tools/validate/` のテスト対象
- **学習教材**: 新しいコントリビューターへの説明用

## 重要な注意

サンプルファイルの**本文・判例情報・英訳**は、フォーマットを示すための代表例であり、Phase 1着手時には e-Gov 法令API・JLT-DB・裁判所Webサイトから**再取得・再検証**したうえで `data/phase1-police/` 配下に正式版を配置すること。

サンプルそのものを `data/` にコピーして使うのは禁止。

## サンプル追加のガイドライン

- フォーマットの新しい側面を示せる場合のみ追加(冗長な複数サンプルは避ける)
- サンプルもスキーマ検証を通過すること
- サンプルであることを `## 注記` セクションで明示
