### 重要判例の候補

警察実務で参照される判例は特に多い. Phase 1 では以下のテーマ別判例集の網羅を目標とする:

- 任意取調の限界 (最決 昭和51年3月16日 等)
- 任意同行・任意捜査の方法 (最決 昭和55年9月22日 等)
- 違法収集証拠の排除 (最判 昭和53年9月7日 等)
- 自白の任意性 (最判 昭和23年7月29日 以降の累積)
- GPS 捜査 (最大判 平成29年3月15日)
- 取調の可視化制度

各判例は出典 (最高裁判所 Web サイト) を必ず確認してから追加すること.
