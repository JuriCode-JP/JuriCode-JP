## 警察関連法令を扱う上での注意

- 警察組織法と警察官の職務権限を扱う条文は、運用解釈が判例・通達で大きく形作られている。**判例リンクの密度**がこのPhaseの最大の差別化要素となる。
- 警察庁訓令・通達は法令ではないが、運用上重要。これらは別レイヤー(将来仕様)で扱う。
- 個別事案への適用は扱わない(JuriCode-JPは法令データ基盤であり、法律相談ツールではない)。
