## なぜ憲法を含めるか

JuriCode-JP の四本柱 (警察 / 自治体 / 民事実務 / 税法) を貫く**上位規範**として、
憲法のオープンな構造化は次の用途で価値がある:

1. **警察柱**: 刑訴法・警察法・警職法の運用根拠は憲法第 13・31・33-38 条 (適正手続)。判例引用の合理性確保
2. **自治体柱**: 第 92-95 条 (地方自治) が地方自治法体系のルートノード
3. **民事実務柱**: 第 13・14・25・29 条 (基本権) が私人間効力論の出発点
4. **税法柱**: 第 30・84 条 (租税法律主義) が国税通則法の起点
5. **対外発信**: 公定英訳 (Japanese Law Translation DB) が比較的整備されており、英訳併記の初期実装に向く
