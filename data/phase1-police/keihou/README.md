### 取得・検証手順 (1 条あたり)

1. `tools/fetch-egov` で e-Gov 法令 API v2 から条文 XML を取得
2. `tools/parse` (ja-law-parser 連携) で構造化
3. `tools/shared` の Pydantic IR にマッピング
4. 公定英訳 (日本法令外国語訳DB) があれば併記、なければ `translation_status: draft`
5. 主要判例 (最高裁判所 Web サイト or 判例秘書 = 確認のみ) をリンク
6. `tools/validate/validate-file.py` で検証
7. `last_verified` 日付を更新してコミット (1 条 = 1 コミット)
