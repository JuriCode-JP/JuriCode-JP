## Phase 2 のターゲットユーザー

- **中小企業の法務・コンプライアンス担当**: 取締役・監査役の責任、株主総会、組織変更などの参照基盤
- **司法書士・行政書士**: 商業登記実務での条文照合
- **ベンチャー支援機関**: スタートアップへの設立・ガバナンス支援
- **税理士柱との接続**: 会社法上の計算書類規定 (第 431 条以下) と法人税法を横断する RAG クエリ

## なぜ Phase 1 中に先取りしたか

1. **bulk-ingest.py の汎用性テスト**: phase ディレクトリを跨ぐ orchestrator が正しく動くか確認
2. **Recall ベンチマークのスケールテスト**: コーパスを 1.85 倍に拡張しても retrieval 精度が落ちないかを実証 → 結果: Recall@3 = 100% を維持 (8-query eval-set)
3. **Phase 2 着手障壁の解消**: 構造化済データが既にあれば、Phase 2 期間 (2027-2028) では判例リンク・英訳整備に集中できる
