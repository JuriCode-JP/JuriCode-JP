## ソース別の詳細

### 1. 法令本文 — e-Gov
- 取得元: e-Gov 法令検索 API。`source_url` に当該法令の e-Gov URL を必須記載。
- 本文は**完全コピー・非改変**（CLAUDE.md §4.1）。`verify.py` の round-trip hash で e-Gov 原文との同一性を機械検証。

### 2. 法令の英訳 — 法務省 JLT（PDL1.0）
- 出所: 日本法令外国語訳データベースシステム（https://www.japaneselawtranslation.go.jp/ ）。
- 規約原文: 「コンテンツは権利表記なき限り**公共データ利用規約（第1.0版）＝PDL1.0**が適用」「**引用、複製、転載について…引用し、複製し又は転載して差し支えありません**」。
- **出典表記例**: 「出典：日本法令外国語訳データベースシステム（法務省）」。
- 注意: 翻訳は**正文ではなく参考資料**。最終改正版でないものや「**暫定版**」あり。`translation_status` と版・freshness を明記。**ロゴマーク／キャラクターは PDL1.0 対象外**。
- 公式訳が無い法令は自前訳を `draft` / `community` で掲載し、機械翻訳は `machine_translated: true`（CLAUDE.md §4.3）。

### 3. 裁決 — 国税不服審判所（PDL1.0）
- 公表裁決の本文・要旨は PDL1.0。出典明示で複製・転載可。税務ドメインの否認リスク等のデータ土台。

### 3.1 タックスアンサー — 国税庁（PDL1.0 準拠・政府標準利用規約）
- 出所: 国税庁「タックスアンサー（税金の相談 Q&A）」（https://www.nta.go.jp/taxes/shiraberu/taxanswer/ ）。
- 国税庁サイトは**政府標準利用規約準拠（CC BY 4.0 互換・PDL1.0 同等）**。出典明示で**複製・翻案・再配布（商用含む）可**。
- **出典表記例**: 「出典：国税庁タックスアンサー」。IR では `license: cc-by-jp-nta` / `attribution: 国税庁タックスアンサー` で機械可読に保持。
- タックスアンサーは Q&A 解説文であり e-Gov 法令のような本文非改変 round-trip の対象ではない。`source_url` に当該コードの nta.go.jp URL を必須記載。content 画像（計算表・フローチャート）は**改変せず絶対 URL で参照**（本文に `![alt](URL)` として引用）。
- glossary は法令表（`law_id` 前提）ゆえ taxanswer 略称（`hojin-taxanswer`（法人税）・`sozoku-taxanswer`（相続・贈与、FU-527）・`shohi-taxanswer`（消費税、FU-528）・`gensen-taxanswer`（源泉所得税、FU-529）・`shotoku-taxanswer`（所得税、FU-530）・`joto-taxanswer`（譲渡所得、FU-531）・`inshi-taxanswer`（印紙税・その他の国税、FU-532）等）は非該当。data-source として本ページで記録する。

### 4. 判例本文 — courts.go.jp（著作権法13条）
- **著作権法13条3号**: 裁判所の判決・決定・命令は著作権の目的とならない＝パブリックドメイン。出典明示で引用・転載可。
- 出典は裁判所の permalink、または掲載誌の巻・号・頁。URL は追加時に実在確認（CLAUDE.md §3.5）。

### 5. 判例の要旨 — courts.go.jp（タイト・自前要約）
- courts.go.jp 規約原文: 「当サイトの内容は…**著作権法上認められている利用**を行うことができます。…裁判所に無断で**改変を行うことはできません**」「リンク設定は原則自由」。
- 要旨／判示事項は**調査官等の編集物**で、判決本文（13条）とは別。**逐語コピー・翻訳・MIT 再配布は不可**。
- **採用パターン（自前要約＋公式リンクの併置）**: `cases` の `summary_ja` / `summary_en` は**JuriCode-JP が判決等を基に独自に記述**する（courts.go.jp の要旨を引き写さない）。あわせて当該判例の **courts.go.jp ページ（公式の裁判要旨・判決本文の所在）への `url` リンク**を併置し、公式要旨は**複製せずリンクで参照**する。読者は自前要約で把握し、一次資料へワンクリックで辿れる。
- **引用の選択肢**: 公式要旨を**短く・出典明示で引用**することは、著作権法32条の引用要件（出所明示・主従関係・明瞭区別・必要な範囲）を満たす限り可。**不可なのは「公式要旨の丸写しをデータ本体にする」こと**。
- **リンクの条件（courts.go.jp 規約）**: (1) 裁判所と特別な関係があると誤解させるコメントを付さない（出典リンクとして淡々と置く）。(2) トップページ以外へのリンクは「裁判所Webサイト（courts.go.jp）へのリンク」である旨を明記する。
- リンク＋書誌メタデータ（裁判所・年月日・citation・事件番号）は自由。
- ※ 将来、courts.go.jp の要旨を英訳して再配布する場合は、口頭でなく**書面確認＋法務確認**が必要。

## 二層ライセンスの考え方
- **構造化レイヤー（スキーマ・タグ・ツール・自前要約・自前訳）= MIT**。
- **取り込んだ各コンテンツ = それぞれの出所ライセンス**（法令本文=e-Gov／英訳=PDL1.0／裁決=PDL1.0／判例本文=13条PD）を保持し、**出典を明示**する。
- PDL1.0 は帰属表示が条件なので、英訳・裁決を MIT に“再ライセンス”はしない。出典付きで PDL1.0 のまま収録する。
