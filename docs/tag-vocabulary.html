RAG の Chunk Augmentation で検索精度を上げるため、自由文字列ではなく統制された語彙を使う。
関連: format-spec.html / ir-spec.html / glossary.md

1. なぜ統制語彙が必要か

RAG の精度向上には Chunk Augmentation(チャンクへのメタタグ付与)が決定的に効く。tags: を自由文字列のままにすると:

  • 同義語の揺れ(「正当防衛」「緊急行為」「違法性阻却」)で検索性が落ちる
  • 上位下位概念の整理ができない
  • ベクトル検索とキーワード検索のハイブリッドが弱くなる

JuriCode-JP は 5 種類のタグカテゴリを定義し、各条文に最低 1 個ずつ付与する規約を設ける。

2. タグ規約

2.1 命名規則

  • 文字種: 日本語(漢字・ひらがな・カタカナ)、または ASCII 小文字 + ハイフン
  • 空白: なし(正当防衛 は OK、正当 防衛 は NG)
  • 接頭辞: カテゴリを示すプレフィックスは付けない(複合タグは例外)
  • 複数語: ハイフン区切り

2.2 1 つの条文に必須カテゴリ

カテゴリ説明必須例
A. フェーズJuriCode-JP のロードマップ上の位置づけ✅phase1-police
B. 法分類法令の大分類(刑事/民事/商事/憲法/行政)✅刑事法
C. 概念条文が扱う中核概念✅(1 個以上)正当防衛
D. 手続き関連する手続き種別任意捜査
E. ステークホルダー主体・対象任意警察

3. カテゴリ別 統制語彙集

A. フェーズタグ必須

phase1-police Phase 1(2026-2027、警察関連法令)
phase2-civil Phase 2(2027-2028、民事・商事)
phase3-all Phase 3(2028-2030、全領域)

B. 法分類タグ必須・1 個
タグ意味Phase 1 対象?
憲法憲法(将来)
刑事法刑法・刑訴法・特別刑法✅
警察法警察法・警職法✅
民事法民法・特別民法Phase 2
商事法商法・会社法Phase 2
行政法行政手続法・行政組織法Phase 3
労働法労基法・労働契約法Phase 3
税法所得税法・法人税法Phase 3
C. 概念タグ必須・1 個以上

C-1. 刑法総則関連

罪刑法定主義 因果関係 故意 過失 責任能力 違法性阻却事由 責任阻却事由 正当防衛 緊急避難 過剰防衛 未遂 既遂 共犯 共同正犯 教唆 幇助 刑罰 量刑

C-2. 刑法各論関連

生命に対する罪 身体に対する罪 自由に対する罪 名誉に対する罪 財産に対する罪 公共危険罪 公務妨害 偽証罪

C-3. 刑事訴訟法関連

捜査手続 逮捕 勾留 取調べ 黙秘権 弁護人選任権 捜索差押 通信傍受 供述調書 公訴提起 公判手続 自白 証拠能力 伝聞法則 裁判員制度 控訴 上告 再審

C-4. 警察法・警職法関連

警察組織 警察の責務 職権発動 職務質問 保護 避難等の措置 武器使用

C-5. 横断的概念

人権 適正手続 比例原則 令状主義 任意性 強制処分

D. 手続きタグ任意

捜査 逮捕 勾留 取調べ 送致 起訴 公判 判決 執行 更生

カテゴリ C との重複があるが、C は概念ラベル・D は手続きフローのラベルとして使い分ける。

E. ステークホルダータグ任意

市民 被疑者 被害者 弁護人 警察 検察 裁判官 裁判員 行政機関 海外

4. 付与例

例 1: 刑法 36 条(正当防衛)

  - 市民                   # E. ステークホルダー(防衛の主体)

5. 運用ルール

5.1 タグ追加のプロセス

  1. 既存語彙で表現困難な概念に遭遇 → Issue で議論
  2. 合意形成 → 本ファイル(docs/tag-vocabulary.md)に追記する PR
  3. PR 通過後に対象条文の tags: に追加

5.2 検証ルール(tools/validate/)

  • ✅ 各条文に カテゴリ A・B・C のタグが最低 1 個ずつある
  • ✅ すべてのタグが本ファイルの統制語彙集に存在する
  • ⚠️ 警告: 1 条文のタグが 10 個超(過剰)、または 3 個未満(過少)

5.3 検索・RAG での活用

メタタグフィルタ検索: ユーザが「正当防衛について教えて」と質問 → 正当防衛 タグを持つ条文だけに絞り込み → そこから cos 類似度で検索 → 精度向上
Top-K 戦略: 同じタグを持つ条文を関連条文として複数同時取得
カテゴリ別ルーティング(Self-route): 質問内容から推定したカテゴリでフィルタした上で検索

6. RAG 動画からの示唆

動画の指摘本仕様での対応
「メタタグ付与で検索精度が上がる」カテゴリ A-E の 5 軸タグ体系
「カテゴリ別ルーティング(Self-route)」カテゴリ B(法分類)が一次フィルタとして機能
「Chunk-to-Question」article_caption(「(正当防衛)」等)が事実上これに相当
「テストデータ重要」デジタル庁 lawqa_jp とのマッピングは lawqa-jp-integration-notes.html
